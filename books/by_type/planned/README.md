# Planned Books By Type

这里记录按成书形态预留的新卷入口。真实目录创建前，不在这里放正文和实验。

计划中的卷：

- 计算机组成原理原理卷：真实目录建议 `books/computer-organization-volume-1/`。
- 计算机组成原理实践与代码卷：真实目录建议 `books/computer-organization-practice/`。
- 操作系统原理卷：真实目录建议 `books/operating-systems-volume-1/`。
- 操作系统实践与代码卷：真实目录建议 `books/operating-systems-practice/`。
- 网络原理卷：真实目录建议 `books/computer-networks-volume-1/`。
- 网络实践与代码卷：真实目录建议 `books/computer-networks-practice/`。

命名规则：

- 真实目录可以保留机器友好的英文 book id。
- 导出到手机和微信读书的 PDF/EPUB 文件名必须使用中文书名，并避免短横线、加号和通用 `main` 文件名。
- 实践与代码卷单独成册，不把大量完整源码塞进原理卷正文，也不把实践任务和源码阅读拆成两本互相跳转的书。
