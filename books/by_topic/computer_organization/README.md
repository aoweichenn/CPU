# Computer Organization

计算机组成原理方向预留目录。建议以后按以下方式拆卷：

- 原理卷：数据表示、指令集、数据通路、控制器、流水线、Cache、内存层次、I/O 和总线。
- 实践卷：从可运行模拟器、汇编样例、Cache 实验、流水线 hazard 到性能报告。
- 源码卷：如果模拟器或验证工程足够大，单独成册收录完整代码、测试和 benchmark。
