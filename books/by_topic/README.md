# Books By Topic

这个目录按知识领域整理入口，不替代真实书籍目录。真实书籍仍放在
`books/<book-id>/`，这里用稳定入口帮助查找、规划和扩展后续系列。

领域划分：

- `cpp_language/`：C++ 语言、标准库、工程组织、构建测试和性能基础。
- `algorithms/`：算法刷题、数据结构、复杂度、面试表达和 C++ 题解。
- `computer_systems/`：从 C++ 到计算系统系列，覆盖程序、二进制、CPU、并发、运行时和系统性能。
- `ai_computing/`：AI 模型、张量、算子、量化和本地推理引擎。
- `computer_organization/`：后续计算机组成原理卷预留。
- `operating_systems/`：后续操作系统卷预留。
- `computer_networks/`：后续网络卷预留。
