# Computer Networks

计算机网络方向预留目录。建议以后按以下方式拆卷：

- 原理卷：分层模型、链路、IP、路由、TCP/UDP、拥塞控制、DNS、HTTP、TLS 和可观测性。
- 实践卷：从 socket、抓包、协议状态机、可靠传输、HTTP 服务到压测和故障注入。
- 源码卷：如果网络协议栈或服务框架成规模，单独成册收录完整代码、测试和 benchmark。

真实目录建议：

- `books/computer-networks-volume-1/`
- `books/computer-networks-practice/`
- `books/computer-networks-source/`
