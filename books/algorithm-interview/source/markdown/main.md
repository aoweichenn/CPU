# 算法刷题与 C++ 面试教材

副标题：三个月建立可复用的算法推理、C++ 实现和面试表达能力。

本书面向准备互联网、基础软件、系统、后端、客户端、AI 工程岗位笔试和面试的读者。它不把刷题理解成背题解，而是训练一套稳定的推理方法：先写得出，再证明对，再解释为什么能优化，最后用 C++ 写出边界清楚、复杂度可控、面试可讲的代码。

## 学习主线

本书每个算法都按同一条路线讲：

1. 先读题，拆出输入、输出、约束和边界。
2. 写暴力解，明确枚举对象和复杂度。
3. 找重复工作，判断能不能用缓存、单调性、局部状态或数据结构消掉。
4. 写优化解，说明不变量为什么成立。
5. 用 C++ 实现，解释容器、迭代器、拷贝、溢出、边界。
6. 做至少三道同类题，确认不是只背了一个题。
7. 写复盘，把模板、坑点和变形条件记录下来。
