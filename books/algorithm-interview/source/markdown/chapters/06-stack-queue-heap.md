# 第 6 章：栈、队列、单调结构和堆

栈、队列和堆都不是单纯容器题，而是候选集合的组织方式。栈处理最近关系，队列处理层次扩展，堆处理动态极值，单调结构处理“被新元素淘汰的旧候选”。

## 栈

有效括号是最基本的栈题。栈里保存尚未匹配的左括号。遇到右括号时，必须和栈顶匹配。这背后的性质是：最近打开的括号必须最先关闭。

单调栈进一步利用“淘汰”。每日温度中，栈里保存还没找到更高温度的下标，并且温度从栈底到栈顶单调递减。新温度到来时，它会解决所有比它低的旧下标。

关键问题：栈里存下标还是值？如果题目要距离、宽度、位置，通常存下标。

## 队列和 BFS

BFS 的核心不变量是：队列中按距离从小到大处理节点。处理一层时，队列里的所有节点距离起点相同。腐烂橘子是多源 BFS：一开始所有腐烂橘子同时入队，每一层代表一分钟。

多源 BFS 模板：

```text
把所有源点入队
distance = 0
while queue 非空:
    处理当前层所有节点
    扩展未访问邻居
    distance += 1
```

## 堆

堆适合动态维护最大或最小元素。前 K 个高频元素可以统计频次后，用小顶堆维护 K 个候选；数组中第 K 大元素可以用大小为 K 的小顶堆，也可以用快速选择。

堆的限制是不能高效删除任意元素。滑动窗口最大值如果用堆，需要懒删除过期下标；用单调队列更直接。

## 单调队列

滑动窗口最大值中，队列保存可能成为最大值的下标，且对应值单调递减。新元素进入时，把队尾所有不大于它的元素弹出，因为这些旧元素更小、还更早过期，不可能再成为答案。

## 本章题单

栈：有效括号、最小栈、逆波兰表达式求值、基本计算器。

单调栈：每日温度、下一个更大元素、柱状图最大矩形、最大矩形。

队列/BFS：二叉树层序遍历、腐烂的橘子、打开转盘锁、完全平方数。

堆：数组中的第 K 个最大元素、前 K 个高频元素、合并 K 个升序链表、数据流的中位数。

复盘要求：写清楚候选什么时候入、什么时候出、为什么被弹出的候选永远不会再成为答案。
