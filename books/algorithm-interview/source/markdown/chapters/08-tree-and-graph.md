# 第 8 章：树、图、BFS 和 DFS

树和图题的第一难点是建模。题目未必直接给你图，但只要有“状态”和“状态之间的一步转移”，就可以把它看成图。网格题的格子是节点，相邻方向是边；课程表的课程是节点，先修关系是有向边；单词接龙的单词是节点，一次变化是边。

## BFS：最短步数和层次扩展

BFS 适合无权图最短步数、层序遍历、多源扩散。腐烂橘子的暴力想法是每一分钟扫描整个网格，看哪些新鲜橘子会腐烂。这样会重复扫描大量空格和已经稳定的格子。

优化方法是多源 BFS：初始所有腐烂橘子入队，只从变化边界向外扩散。每个橘子最多入队一次，总复杂度 `O(mn)`。

核心不变量：处理第 `t` 层时，队列里的节点都是第 `t` 分钟刚能被影响到的节点。

## 岛屿数量：连通块

岛屿数量的本质是统计无向图连通块。每个陆地格子是节点，相邻陆地之间有边。每遇到一个未访问陆地，就从它开始 BFS，把整个连通块标记掉，答案加一。

C++ 实现里可以把网格按值传入函数，然后原地改副本作为访问标记。这样代码简单，也不修改调用者。

## DFS 与显式栈

递归 DFS 写起来短，但 C++ 在线评测中深度过大可能栈溢出。本书会优先展示 BFS 或显式栈。显式栈和递归本质一样，只是把调用栈变成自己管理的数据结构。

## 拓扑排序

课程表是有向图环检测。入度为 0 的点表示当前没有前置依赖，可以先学。不断移除入度为 0 的点，如果最后处理了所有课程，说明无环；否则存在环。

## 本章题单

树：二叉树层序遍历、最大深度、对称二叉树、路径总和、二叉搜索树验证、最近公共祖先。

网格图：岛屿数量、腐烂的橘子、被围绕的区域、太平洋大西洋水流问题。

图：克隆图、课程表、课程表 II、单词接龙、除法求值。

复盘要求：每题先写“节点是什么、边是什么、访问状态是什么、队列或栈里保存什么”。
